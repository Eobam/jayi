# jayi

## What is this?
This is a Python script which acts kinda like Jay (east coast rm)

## How do I use it? 
Just download the .app file (only works on Mac) from the Github release, run it, and talk to "jay"!

## Are there any... special prompts?

yes :3.. Try adding (not case sensitive) "HCB", "unc", "blueprint", "jay don't", or "jay dont" into your prompt, and see what happens.
